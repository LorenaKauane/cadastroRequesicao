# cadastroRequesicao
CRUD cadastro de requesições além dos estudos de pesquisa generica.

# Funcionalidades
 * Cadastro de usuario
 * Cadastro de requesição 
 * Campo de busca (Generico utilizando querydsl)

# Tecnologias utilizadas e frameworks 
 * Java EE
 * Maven
 * Spring boot
 * Spring data JPA
 * Spring rest
 * Spring querydsl
 * Postgresql
 * Angular
 * Bootstrap
